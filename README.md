# Исполнитель команд для светодиодной матрицы (triangle Matrix)![Table in PWR](ca104887_a946_4308_b35b_d2a5631e7dea.jpg) 

## 📖 Общее описание

Этот проект представляет собой интерпретатор команд для микроконтроллера ATmega328P, который управляет светодиодной матрицей 10×10. Исполнитель загружает текстовый файл с командами, преобразует их в байт-код и последовательно выполняет, управляя светодиодами и реагируя на ввод с клавиатуры (через UART).

## 🎯 Назначение

Исполнитель позволяет:
- Управлять 100 светодиодами (матрица 10×10)
- Читать нажатия клавиш через UART
- Выполнять условные переходы на основе введённых клавиш
- Создавать задержки для анимаций

## 🧠 Архитектура

### 1. Макросы и константы
- `name.asm` — содержит определения для всех 100 светодиодов:
  - `R0C0` = 0, `R0C1` = 1, ..., `R9C9` = 99
- ОПКОДЫ команд:
  ```asm
  OP_LED_ON   = 0x01
  OP_LED_OFF  = 0x02
  OP_RDG_KEY  = 0x03
  OP_CND      = 0x04
  OP_DELAY    = 0x05
  OP_END      = 0xFF
